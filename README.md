## 项目描述
实现简易购物车功能，可以完成添加、删除、结算等功能。购物车会实时显示物品数量，点击购物车按钮可以看
到具体物品，同种物品数量，并且可以减少或者删除，点击结算将对每种物品和所有物品分别结算。<br><br>
